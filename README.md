# Spam Message Detection using Machine Learning

A machine learning project that classifies text messages as **Spam** or **Ham** using Natural Language Processing (NLP).

The project explores the complete machine learning workflow, including data cleaning, exploratory data analysis, feature engineering with TF-IDF, model comparison, evaluation, and model deployment by saving the trained classifier.

---

## Dataset

The project uses the **SMS Spam Collection Dataset**, which contains over 5,500 labeled SMS messages.

Classes:
- Ham (legitimate)
- Spam

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Multiple Machine Learning Models
- Model Comparison
- Confusion Matrix
- Classification Report
- Error Analysis
- Model Saving with Joblib
